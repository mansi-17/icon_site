About The Event

Are you a programmer? Do you want to test your coding skills in a fun and challenging way? Then look no further as the ICON team has brought an exceptionally amusing game - Blind coding, to scratch that coder's itch! The goal is pretty simple: you have to write the entire code in one go without running it, all the while keeping the number of lines to a minimum.

Register here



Event format:

  • Round 1: Problem Statement


    No. of participants: 16 Problem Level: Easy Duration: 20 minutes

  • Participants then will be divided into breakout rooms such that, there will be one participant and a volunteer in one breakout room.

  • Round 2:


    No. of participants: 8 Problem Level: Medium Duration: 30 minutes

  • Round 3: Final Round


    No. of participants: 4 Problem Level: Hard Duration: 45 minutes


Event rules:

  • Member should be college student.
  • For valid participation the member should upload their ID card scanned copy during the registration.
  • If the participant doesn't have an ID card, then fee receipt of the current year can be uploaded instead.
  • Code must be written in C++ programming language.
  • Maximum number of lines would be specified in which the participant would need to finish the code.
  • Participant has to replicate the output provided in the question. Failure to do so or any runtime error would lead to elimination.
  • Using backspace would lead to a penalty of 2 minutes. Breaking the rule more than 5 times would lead to disqualification per round.
